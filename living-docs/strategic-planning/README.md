## How to Use This

These documents are designed to be living plans -- revisited quarterly and updated as circumstances change. They integrate the personal (partnership, life design, where to live) with the professional (projects, funding, movement work) because for Syd and Eslerh, these are not separate.

Each document includes:
- Current state assessment
- Strategic questions to revisit
- Decision frameworks for key choices
- Concrete next steps with timelines

## Implementation Guardrails for Builders

These are non-negotiable principles for anyone building or configuring the Live with Syd tech stack.

### Prioritize

- Open-source or privacy-first tools
- GDPR-friendly data handling
- Minimal data collection
- No resale or third-party sharing of participant data
- Clear consent prompts
- Participant ability to opt out of recordings
- Easy content removal on request

### Avoid

- Zoom or Meta platforms
- Tools with ambiguous media rights clauses
- Forced logins or invasive tracking
- Complex onboarding flows that create friction

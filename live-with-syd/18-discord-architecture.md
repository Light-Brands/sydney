## Server 1 — Regenerative Capital Flows

**Server Purpose:**
A learning, ideation, and collaboration hub for unlocking capital for grassroots and bioregional regeneration.

**Tone:** Strategic, experimental, grounded, hopeful
**Growth Target:** 100-500+ members over time
**Moderation Level:** Medium
**Use Case:** Ongoing dialogue, resource-sharing, project matchmaking, funder-organizer bridging

### Category 0 — Welcome & Orientation

| Channel | Purpose |
|---------|---------|
| `#welcome` | What this server is, who it's for, how to get started, link to Notion hub |
| `#start-here` | How to participate, participation roles, how to propose a topic, how to pitch a project, link to registration + speaker intake |
| `#community-agreements` | Agreements, consent norms, anti-promo policy, conflict pathway *(Read-only)* |
| `#announcements` | Upcoming Monday sessions, guest speakers, special events *(Mods only)* |

### Category 1 — General Field

| Channel | Purpose |
|---------|---------|
| `#introductions` | Member intros |
| `#general-chat` | Open discussion |
| `#gratitude` | Appreciation and acknowledgement |
| `#requests-for-support` | Ask for help |
| `#offers-of-support` | Offer help |

### Category 2 — Capital Flows Core

| Channel | Purpose |
|---------|---------|
| `#capital-announcements` | Key updates *(Mods only)* |
| `#capital-discussion` | General capital flows dialogue |
| `#capital-ideation` | New ideas and brainstorming |
| `#capital-resources` | Links, tools, reading |
| `#capital-case-studies` | Real-world examples |
| `#capital-projects` | Projects seeking funding |
| `#capital-funders-lounge` | Funder-specific discussion *(role-gated)* |
| `#capital-commitment-pooling` | Pooled funding experiments |
| `#capital-data-impact` | Metrics, storytelling, dashboards |

### Category 3 — Collaboration

| Channel | Purpose |
|---------|---------|
| `#collaboration-requests` | Open asks |
| `#working-groups` | Active working group coordination |
| `#volunteer-opportunities` | Ways to contribute |
| `#feedback-and-ideas` | Server and ecosystem feedback |

### Category 4 — Archives

| Channel | Purpose |
|---------|---------|
| `#session-replays` | Links to past sessions *(Read-only)* |
| `#session-notes` | Notes and summaries |
| `#resource-library` | Curated resources |

### Category 5 — Private (Mods Only)

| Channel | Purpose |
|---------|---------|
| `#mod-hub` | Moderator coordination |

| `#ecosystem-strategy` | Strategic planning |
